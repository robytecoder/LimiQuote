## Requirements

- _Docker_ and _docker-compose_
- _bash_

## Quickstart

- build with `docker-compose build`
- run with `docker-compose up`
- seed the initial data by running `./scripts/seed.sh` in another terminal (with bash)
- browse _http://localhost:3000_
